## Overview

**Echoes Of Syllis** is a sci-fi action experience set in a mysterious world shaped by forgotten technology, ancient ruins, and hostile forces.

The project is currently in active development using the Unity Engine as part of an ongoing game development journey.

---

## Features

- Player movement system
- Combat mechanics
- Enemy AI
- Health system
- Score system
- Collectibles / pickups
- Main menu UI
- Atmospheric sci-fi environment
- Progressive gameplay systems
